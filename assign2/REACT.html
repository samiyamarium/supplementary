INTRODUCTION

React is a javascript libraray. It offers modularity.A developer often needs to use reusable components and mostly like to build a single page web.In this case , React is an Ideal choice!! It is used to build User Interface componenents..

STRUCTURE

React functionality is divided into two main parts for User Interface: 1.Components | 2. Props
1.Components: They are functions that return HTML using render() method.
2.Prpos: It is the input that we pass to componenets, just like, function and properties in TypeScript!!
**Components should start with a Capital letter!! It treats lowercase components as DOM tags ∔
